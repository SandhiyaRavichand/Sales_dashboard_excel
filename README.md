# Automated Sales dashboard (Excel 2007)

This project demonstrates how to build a **fully automated sales performance dashboard** using **Excel 2007**, with data cleaning, PivotTables,charts, macros, and basic VBA scripting.It is designed as a beginner-friendly analytics project for reporting or analyst roles.

----

## Features

- Sales data summary by **region, product, month, and salesperson**
- Interactive **PivotTables** and **charts**
- Clean and professional **dashboard layout**
- One-click **Refresh Dashboard** button (via macro)
- Uses **VBA** to automate refresh and navigation

----

## Dataset

A sample sales dataset with the following columns:
- 'Date'
- 'Salesperson'
- 'Product'
- 'Sales Amount'

----

## Dashboard Sections

- **Sales by Salesperson**
- **Monthly Sales Trend**
- **Sales by Product**
- KPIs and total sales cards

----

## Automation

### Macro: 'RefreshDashboard'
- Refreshes all PivotTables
- Navigates to the dashboard sheet
- Shows a confirmation message box

**Macro Shortcut:** 'Ctrl + Shift + R'

----

## How to use

1. Open the 'Sales_dashboard.xlsm' file.
2. Enable macros when prompted.
3. Click the **Refresh Dashboard** button to update the dashboard.

----

## Tools Used

- Microsoft Excel 2007
- PivotTables & PivotCharts
- Macros (via Developer tab)
- VBA scripting(Alt + F11)
